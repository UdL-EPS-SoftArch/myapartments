MyApartments

MyApartments is an application designed to facilitate the management and search of apartments, offering an intuitive interface for both owners and clients.

Advertisements

This screen displays a list of registered advertisements, showing their details. It features buttons to edit or delete each advertisement. Also, by clicking an advertisement you can access to their details page.

Apartments

This screen displays a list of registered apartments, showing their details. It features buttons to edit or delete each apartment. Also, by clicking an apartment you can access to their details page.

Rooms

This screen displays a list of registered rooms, showing their details. It features buttons to edit or delete each room.

Documentation

This screen displays the API documentation, showing the available endpoints and their details.
