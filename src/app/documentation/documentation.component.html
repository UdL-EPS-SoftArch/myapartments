MyApartments

Description

MyApartments application is a project developed by 4th-year software engineering students from the University of Lleida (UdL).

Its purpose is to create a rental portal exclusively for students, allowing them to browse available apartment listings and their respective rooms, request a visit, track the status of their requests, leave comments, and more.

On the other hand, apartment owners can manage all their listed properties, oversee advertisements and occupancy statuses, schedule visits, and perform additional tasks.

Technologies and Architecture

MyApartments project is built using a robust architecture and modern technologies to ensure scalability and maintainability:

  • Frontend: Developed with Angular, using TypeScript for a component-based architecture (MVC) and npm for dependency management. Cypress and Karma are used for testing (end-to-end and unit tests, respectively).
  • Backend: Built with Spring Boot in Java, leveraging JPA for data persistence. The RESTful API includes HATEOAS for dynamic resource navigation.
  • Containerization: Docker is used for creating containerized environments and deploying the application.

The architecture follows a clear client-server model, separating frontend and backend concerns. Both layers implement the MVC pattern for structured development. Additionally, the repository pattern is utilized for database operations, ensuring clean and modular code.
